## About

This project is demo of using [`freeapi`](https://freeapi.app/). 

This is auth project to demonstrate how to use freeapi to get data and use it in your project.

## What this project does?

- Next.js
- Tailwind CSS
- FreeAPI
- Login
- Register
- protected routes
- Logout
- Profile
- forgot password
- reset password
- verify email
